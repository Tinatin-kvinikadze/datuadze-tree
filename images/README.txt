# images folder

Place your family photos here.

## Gallery photos
Name them: photo1.jpg, photo2.jpg, photo3.jpg, photo4.jpg, photo5.jpg, photo6.jpg

## Family member photos (for the cards)
Name them anything descriptive, for example:
- father.jpg
- mother.jpg
- grandfather-paternal.jpg
- grandmother-maternal.jpg
- etc.

Then open js/script.js and set the `photo` field on the right family member,
for example:  photo: 'images/father.jpg'

Supported formats: JPG, PNG, WebP
Recommended size: at least 200x200 pixels for member cards, 800x600 for gallery
